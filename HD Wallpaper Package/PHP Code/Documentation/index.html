How to Configure Webservice In Your Server?

Live Installation

  2. Then create a database.
  3. Import the database from the db folder.
  4. Aftre creating database configure the connection.php file,which is in the includes folder of your package.
    $serverIp="localhost"; //Depends on hosting, normaly localhost
    $userName="db_user";
    $password="db_pass";
    $dbname="db_name";
